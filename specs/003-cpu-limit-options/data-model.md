## 1. CPU Limiting Entities

### 1.1 CPULimitConfig

Represents user's CPU throttling preferences.

```python
@dataclass
class CPULimitConfig:
    """Configuration for CPU limiting during video processing"""
    enabled: bool = False
    limit_percent: int = 75  # Range: 20-95
    monitor_interval: float = 1.0  # Seconds between CPU checks
    priority_level: str = "normal"  # Options: low, normal, high

    def validate(self) -> tuple[bool, str]:
        """Validate configuration values"""
        if self.limit_percent < 20 or self.limit_percent > 95:
            return False, "CPU limit must be between 20% and 95%"
        if self.priority_level not in ["low", "normal", "high"]:
            return False, "Priority must be low, normal, or high"
        if self.monitor_interval < 0.5 or self.monitor_interval > 5.0:
            return False, "Monitor interval must be between 0.5 and 5.0 seconds"
        return True, ""
```

**Relationships**:
- Stored in `AppState.cpu_limit_config`
- Used by `CPULimiter` to configure monitoring and thread calculation
- Persisted in `settings.json` via `SettingsManager`

**Validation Rules**:
- `limit_percent`: 20 ≤ value ≤ 95
- `monitor_interval`: 0.5 ≤ value ≤ 5.0
- `priority_level`: Must be one of ["low", "normal", "high"]

**State Transitions**:
```
enabled=False → User enables → enabled=True → Monitoring starts
enabled=True → User disables → enabled=False → Monitoring stops
enabled=True → User changes limit_percent → Apply new limit to active processes
```

### 1.2 CPUMetrics

Real-time CPU usage metrics for display.

```python
@dataclass
class CPUMetrics:
    """Current CPU usage metrics"""
    process_cpu_percent: float = 0.0  # This process only
    system_cpu_percent: float = 0.0   # Entire system
    thread_count: int = 0              # FFmpeg threads in use
    target_limit: int = 75             # User-configured limit
    timestamp: float = 0.0             # time.time() of last update

    @property
    def is_within_limit(self) -> bool:
        """Check if current usage is within 5% of target"""
        return abs(self.process_cpu_percent - self.target_limit) <= 5.0

    @property
    def variance_percent(self) -> float:
        """Calculate variance from target limit"""
        return self.process_cpu_percent - self.target_limit
```

**Relationships**:
- Generated by `CPULimiter.get_current_metrics()`
- Displayed in real-time by UI components via callbacks
- Not persisted (runtime-only data)

## 2. Enhanced Settings Entities

### 2.1 ApplicationSettings

Complete settings structure with all categories.

```python
from pydantic import BaseModel, Field, validator

class PerformanceSettings(BaseModel):
    """Performance and resource control settings"""
    cpu_limiting_enabled: bool = False
    cpu_limit_percent: int = Field(75, ge=20, le=95)
    max_parallel_jobs: int = Field(2, ge=1, le=4)
    ffmpeg_priority: str = Field("normal", regex="^(low|normal|high)$")

    @validator('cpu_limit_percent')
    def validate_cpu_limit(cls, v):
        if v % 5 != 0:  # Enforce 5% increments
            raise ValueError("CPU limit must be in 5% increments")
        return v

class OutputSettings(BaseModel):
    """Output file naming and organization settings"""
    naming_pattern: str = "{prefix}{filename}{suffix}"
    prefix: str = ""
    suffix: str = ""
    output_format: str = "mp4"
    create_subfolder: bool = True
    overwrite_policy: str = Field("ask", regex="^(ask|skip|overwrite)$")

    @validator('naming_pattern')
    def validate_pattern(cls, v):
        allowed_vars = ["{prefix}", "{filename}", "{suffix}", "{date}", "{time}", "{profile}"]
        # Check pattern contains only allowed variables
        import re
        vars_in_pattern = re.findall(r'\{[^}]+\}', v)
        if any(var not in allowed_vars for var in vars_in_pattern):
            raise ValueError(f"Pattern can only contain: {', '.join(allowed_vars)}")
        return v

class QualitySettings(BaseModel):
    """Video and audio quality settings"""
    default_profile: str = "Universal Compatible"
    video_bitrate_override: Optional[int] = Field(None, ge=500, le=50000)  # kbps
    audio_quality: str = Field("medium", regex="^(low|medium|high|lossless)$")
    maintain_aspect_ratio: bool = True

class AdvancedSettings(BaseModel):
    """Advanced and experimental settings"""
    custom_ffmpeg_params: str = ""
    logging_level: str = Field("info", regex="^(error|warning|info|debug)$")
    enable_intro_detection: bool = False
    detection_confidence_threshold: float = Field(0.75, ge=0.5, le=0.95)
    keep_temp_files: bool = False

class ApplicationSettings(BaseModel):
    """Complete application settings"""
    version: str = "1.0.0"
    performance: PerformanceSettings = PerformanceSettings()
    output: OutputSettings = OutputSettings()
    quality: QualitySettings = QualitySettings()
    advanced: AdvancedSettings = AdvancedSettings()
```

**Relationships**:
- Managed by `SettingsManager`
- Persisted to `settings.json` on changes
- Loaded into `AppState` on startup
- UI reflects and modifies via `SettingsManager` API

**Validation Rules**:
- All fields validated via Pydantic validators
- Invalid values trigger clear error messages
- Defaults applied for missing fields

## 3. Intro/Outro Detection Entities

### 3.1 DetectionProfile

Stores learned patterns for a series.

```python
@dataclass
class SegmentPattern:
    """Pattern data for intro or outro segment"""
    perceptual_hashes: List[str]  # 64-bit pHash as hex strings
    duration_seconds: float
    start_offset: float = 0.0  # Typical start time in video
    audio_fingerprint: Optional[str] = None  # Optional audio pattern
    sample_count: int = 1  # Number of episodes used to create pattern

    def similarity_score(self, other_hashes: List[str]) -> float:
        """Calculate similarity between this pattern and candidate hashes"""
        if len(other_hashes) != len(self.perceptual_hashes):
            return 0.0

        total_distance = 0
        for p1, p2 in zip(self.perceptual_hashes, other_hashes):
            # Hamming distance between hashes
            h1 = int(p1, 16)
            h2 = int(p2, 16)
            distance = bin(h1 ^ h2).count('1')
            total_distance += distance

        max_distance = len(self.perceptual_hashes) * 64  # 64 bits per hash
        similarity = 1.0 - (total_distance / max_distance)
        return similarity

@dataclass
class DetectionProfile:
    """Complete detection profile for a series"""
    series_id: str  # Unique identifier (hash of series name)
    series_name: str  # Human-readable name
    intro_pattern: Optional[SegmentPattern] = None
    outro_pattern: Optional[SegmentPattern] = None
    confidence_threshold: float = 0.75
    user_corrections: int = 0  # Count of manual corrections
    episodes_analyzed: int = 0
    last_updated: float = 0.0  # timestamp

    def to_dict(self) -> dict:
        """Serialize to dictionary for JSON storage"""
        return {
            "series_id": self.series_id,
            "series_name": self.series_name,
            "intro_pattern": {
                "hashes": self.intro_pattern.perceptual_hashes,
                "duration": self.intro_pattern.duration_seconds,
                "offset": self.intro_pattern.start_offset,
                "audio_fingerprint": self.intro_pattern.audio_fingerprint,
                "sample_count": self.intro_pattern.sample_count
            } if self.intro_pattern else None,
            "outro_pattern": {
                "hashes": self.outro_pattern.perceptual_hashes,
                "duration": self.outro_pattern.duration_seconds,
                "offset": self.outro_pattern.start_offset,
                "audio_fingerprint": self.outro_pattern.audio_fingerprint,
                "sample_count": self.outro_pattern.sample_count
            } if self.outro_pattern else None,
            "confidence_threshold": self.confidence_threshold,
            "user_corrections": self.user_corrections,
            "episodes_analyzed": self.episodes_analyzed,
            "last_updated": self.last_updated
        }

    @classmethod
    def from_dict(cls, data: dict) -> "DetectionProfile":
        """Deserialize from dictionary"""
        intro = None
        if data.get("intro_pattern"):
            ip = data["intro_pattern"]
            intro = SegmentPattern(
                perceptual_hashes=ip["hashes"],
                duration_seconds=ip["duration"],
                start_offset=ip.get("offset", 0.0),
                audio_fingerprint=ip.get("audio_fingerprint"),
                sample_count=ip.get("sample_count", 1)
            )

        outro = None
        if data.get("outro_pattern"):
            op = data["outro_pattern"]
            outro = SegmentPattern(
                perceptual_hashes=op["hashes"],
                duration_seconds=op["duration"],
                start_offset=op.get("offset", 0.0),
                audio_fingerprint=op.get("audio_fingerprint"),
                sample_count=op.get("sample_count", 1)
            )

        return cls(
            series_id=data["series_id"],
            series_name=data["series_name"],
            intro_pattern=intro,
            outro_pattern=outro,
            confidence_threshold=data.get("confidence_threshold", 0.75),
            user_corrections=data.get("user_corrections", 0),
            episodes_analyzed=data.get("episodes_analyzed", 0),
            last_updated=data.get("last_updated", 0.0)
        )
```

**Relationships**:
- Cached in `AppState.detection_profiles: Dict[str, DetectionProfile]`
- Persisted to `detection_profiles/` directory as individual JSON files
- Created/updated by `IntroOutroDetector`
- Used by UI to display detected segments

**Storage Format**:
```
detection_profiles/
├── a1b2c3d4.json  # series_id hash
├── e5f6g7h8.json
└── index.json     # Maps series names to IDs
```

### 3.2 DetectionResult

Result of analyzing a single video file.

```python
@dataclass
class DetectedSegment:
    """A single detected intro or outro segment"""
    segment_type: str  # "intro" or "outro"
    start_time: float  # seconds
    end_time: float    # seconds
    confidence: float  # 0.0 to 1.0
    method: str        # "hash_match", "audio_match", "user_confirmed"

    @property
    def duration(self) -> float:
        return self.end_time - self.start_time

    @property
    def needs_review(self) -> bool:
        """True if confidence below threshold"""
        return self.confidence < 0.75

@dataclass
class DetectionResult:
    """Complete detection result for a video file"""
    file_path: str
    series_id: Optional[str] = None  # None if no matching profile
    intro: Optional[DetectedSegment] = None
    outro: Optional[DetectedSegment] = None
    analysis_duration: float = 0.0  # seconds to analyze
    error: Optional[str] = None  # Error message if detection failed

    @property
    def has_detections(self) -> bool:
        return self.intro is not None or self.outro is not None

    @property
    def all_high_confidence(self) -> bool:
        """True if all detected segments have high confidence"""
        if not self.has_detections:
            return False
        segments = [s for s in [self.intro, self.outro] if s is not None]
        return all(not s.needs_review for s in segments)
```

**Relationships**:
- Generated by `IntroOutroDetector.analyze_video()`
- Displayed in UI for user review
- User confirmations/corrections update associated `DetectionProfile`
- Not persisted (runtime-only, regenerated per analysis)

**State Transitions**:
```
No Profile → Analyze → Low Confidence → User Reviews → Confirms/Corrects → Profile Updated
Existing Profile → Analyze → High Confidence → Auto-Apply → Processing
```

## 5. Data Flow Diagrams

### CPU Limiting Flow

```
User Changes Slider → UI validates → SettingsManager.save()
    ↓
AppState.cpu_limit_config updated
    ↓
CPULimiter.apply_limit(config)
    ↓
Calculate threads → Update FFmpeg command
    ↓
Start monitoring → Every 1s: psutil.Process().cpu_percent()
    ↓
CPUMetrics updated → UI callback → Display updated
```

### Intro/Outro Detection Flow

```
User Enables Detection → Video Added for Processing
    ↓
IntroOutroDetector.analyze_video(file_path)
    ↓
Load/Create DetectionProfile for series
    ↓
Extract frames → Compute pHashes → Match patterns
    ↓
Generate DetectionResult with confidence scores
    ↓
UI displays detected segments → User reviews
    ↓
User confirms/corrects → Update DetectionProfile → Save
    ↓
VideoProcessor applies skip_intro/skip_outro → FFmpeg -ss/-t
```

### Settings Persistence Flow

```
User Modifies Setting → UI validates input
    ↓
SettingsManager.update(category, key, value)
    ↓
Pydantic validation → If valid: update model
    ↓
ApplicationSettings.model_dump() → JSON
    ↓
Atomic write to settings.json (with backup)
    ↓
AppState synchronized with new values
```
